# 如何在 ComfyUI 上設置拓展模型存儲路徑

ComfyUI 支持設置拓展的模型存儲路徑，設置後 ComfyUI 會識別默認的路徑和拓展路徑下的模型文件。推薦大家將模型存儲到 Datadrive，這樣不用每次新增模型配置後製作 snapshot。

---

## 設置拓展模型路徑

以將拓展模型路徑設置為 `/datadrive/comfyui` 為例，我們訪問 My Instance 介面的 HTTP 8888 Port (Jupyterlab 鏈接) 進入實例，新建一個 Terminal 後依次輸入以下指令，複製拓展模型配置文件並創建模型存儲路徑。

```bash
cd /ComfyUI
cp extra_model_paths.yaml.example extra_model_paths.yaml
mkdir /datadrive/comfyui
```

![創建模型存儲路徑](../tutorials-images/comfyui-path/image-20250904135537054.png)

然後雙擊 `/ComfyUI` 目錄下剛剛複製出的 `extra_model_paths.yaml` 文件，將文件內容中 `comfyui` 部分注釋取消，並將 `base_path` 設置為 `/datadrive/comfyui` 即可（注意：修改後按 `Ctrl + S` 保存）。

![修改配置文件](../tutorials-images/comfyui-path/image-20250904135508173.png)

---

## 重啟 ComfyUI

需要重啟 ComfyUI，修改的配置文件才會生效。我們訪問介面的 HTTP Port 8188 進入 ComfyUI 介面。

![訪問 ComfyUI](../tutorials-images/comfyui-path/image-20250905105554414.png)

點擊介面右上角 `Manager -> Restart` 即可重啟 ComfyUI。重啟完成後，位於 `/datadrive/comfyui` 下存儲的模型會被識別並可用。

![重啟後模型生效](../tutorials-images/comfyui-path/image-20250905105632515.png)
